

{\bfseries \doxyref{dcsl\-\_\-messages}{p.}{namespacedcsl__messages}} 

\-This package contains the custom messages used by the \-D\-C\-S\-L \-R\-O\-S packages. \-The beluga\-Input message contains motor commands for the \-D\-C\-S\-L \-Beluga aquatic robots. \-The \-Twist\-Array message is an array of \-Twist messages similar to how a \-Pose\-Array is an array of \-Pose messages.





\-Defined in beluga\-Inpunt.\-msg file.

\-This message has the following properties (name / type)\-:
\begin{DoxyItemize}
\item {\bfseries \char`\"{}vertical\-\_\-motor / int16\char`\"{}} \-: \-Veritcal thurst motor input from -\/255 to 255
\item {\bfseries \char`\"{}thrust\-\_\-motor / int16\char`\"{}} \-: \-Horizontal thrust motor input from -\/255 to 255
\item {\bfseries \char`\"{}servo / int16\char`\"{}} \-: \-Servo motor input that should be from -\/90 to 90
\end{DoxyItemize}





\-Defined in \-Twist\-Array.\-msg file.

\-This message has one property (name / type)\-:
\begin{DoxyItemize}
\item {\bfseries \char`\"{}twists / Twist[$\,$]\char`\"{}} \-: an array of twist messages
\end{DoxyItemize}
